Height: altura

Width: largura

Padding: espaçamento interno, para a borda ficar mais afastada do texto

Margin: espaçamento externo, serve para afastar outros elementos

Border: borda, serve para dar uma borda ao elemento

Outline: borda externa, serve para dar uma borda externa ao elemento, fica fora da borda e dentro do Margin

Display: exibição, serve para exibir o elemento

box-level: é um elemento de bloco, que ocupa toda a largura da tela e quebra a linha exemplo: div, h1-h6, p, main, header, footer, aside, nav, section, article, etc

inline-level: é um elemento de linha, que não quebra a linha e não ocupa toda a largura da tela, exemplo: span, a, img, input, label, button, etc
